# isolated-3d-icon-imagegen

[English](README.md)

画像生成を使って、ゆるくデフォルメされた 3D アイコン風のアイソレ画像を作るための Codex / ChatGPT 向け skill です。

`samples/` に含まれる作例を参照しながら、日用品、ガジェット、食べ物、植物、動物、人物、小さなコンセプトなどを、クリーンな 3D アイコン風のアイソレ画像にします。

## できること

この skill は、以下のような画像生成を支援します。

- アイソレされた 3D アイコン風画像
- 丸みがあり、ほどよく簡略化されたオブジェクト
- 商品、アプリ、EC、マーケティング向けの小さなビジュアル
- サンプルの作風に寄せた画像生成プロンプト

## 用途

- 商品アイコン
- EC 用アセット
- アプリ内イラスト
- SNS やマーケティング向けビジュアル
- オフィス用品、ガジェット、食品、動物、植物、人物風オブジェクトのアイソレ画像

## インストール

Codex に以下のように依頼すると、この GitHub リポジトリから skill をインストールできます。

```text
https://github.com/nin-nin22/isolated-3d-icon-imagegen のスキルをインストールして
```

より明示的に伝える場合は、skill がリポジトリ直下にあることも含めて依頼してください。

```text
https://github.com/nin-nin22/isolated-3d-icon-imagegen のルートにある Codex skill をインストールして
```

インストール時に、Codex が GitHub へのアクセス許可を求める場合があります。インストール後は、新しい skill を読み込むために Codex を再起動してください。

## リポジトリ構成

```text
.
├── SKILL.md          # skill の指示本文
├── agents/
│   └── openai.yaml   # agent 用メタデータ
└── samples/          # 目標作風を定義する参照画像
```

## 使用例

```text
$isolated-3d-icon-imagegen を使って、観葉植物と小さな本の、ゆるくデフォルメされた3Dアイコン風アイソレ画像を作って。
```

```text
$isolated-3d-icon-imagegen を使って、スマートフォンと小さな配送箱のかわいい3Dアイコン風アイソレ画像を生成して。
```

## 補足

このリポジトリには skill の指示と参照サンプルが含まれています。実際の画像は、Codex / ChatGPT 環境で利用可能な画像生成ツールによって生成します。
